## What /ghost does
Takes AI-flavored text and rewrites it so it reads like the player actually wrote it — no send action, just a better draft.

## Steps

1. **Read the text the player provides.** If you don't have a strong sense of their voice yet, check the Status Window / any banked Skill Inventory stories / prior Battle Log messages for tone cues — real sentences they've actually written are worth more than guessing at a generic "professional but human" voice.

2. **Strip the common AI tells**: throat-clearing openers ("I hope this message finds you well", "I'm excited to..."), triplet lists ("not just X, but Y, and importantly Z"), em-dash-heavy rhythm, hedge-everything qualifiers, corporate-speak nouns doing verb jobs, and a suspiciously even, unvaried sentence length. Vary rhythm — short and long sentences mixed, the way people actually write when they're not performing.

3. **Keep the actual content and claims intact.** This is a voice pass, not a fact rewrite — don't add achievements, don't soften or inflate a claim, don't change what's actually being said. If the underlying content itself is thin or unconvincing, say so separately rather than dressing it up with confident-sounding prose that oversells it.

4. **Match register to context** — a cold outreach note, a recruiter reply, and a cover letter each have a different appropriate formality; don't apply one flattening pass to all of them.

5. **Return the rewritten draft** plus a one-line note on the biggest tell(s) you removed, so the player can spot-check the change and learn the pattern for next time. This is a draft for their review and edit — never send it.
